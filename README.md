# Finals Project  
## Bullet Hell Roguelike

A Java-based bullet hell roguelike game developed as a finals requirement.

## How to Run

1. Compile all Java source files:
```bash
javac *.java
```
2. Run the program:
```bash
java GameStarter
```
3. When prompted to input an IP address and port, simply enter:
```bash
123
```
4. The game should run afterward.
